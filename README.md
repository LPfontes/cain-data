# CAIN RPG — Base de Dados e Pacotes de Conteúdo

Este repositório contém todos os dados estruturados em JSON do sistema **CAIN RPG**, incluindo regras base, agendas, blasfêmias, virtudes, equipamentos, marcas de pecado, drifters e os suplementos *Games For Freaks*.

## 📁 Estrutura de Arquivos

- data/agendas.json: Agendas de Exorcista
- data/blasphemies.json: Blasfêmias e Magias
- data/drifters.json: Catálogo de Drifters (Anomalias)
- data/equipment.json: Equipamentos, Kits e Posses
- data/gamesff1.json - gamesff4.json: Suplementos Games For Freaks (Issues 1 a 4)
- data/rules.json: Regras gerais do sistema
- data/sinmarks.json: Marcas de Pecado
- data/sins.json: Pecados
- data/skills.json: Perícias
- data/virtues.json: Virtudes
- data/en/: Traduções dos dados em Inglês

## 🚀 Uso

Esses dados são consumidos diretamente pelo sistema [cain-sheets](https://github.com/LPfontes/cain-sheets) via gerenciador de pacotes de conteúdo (content-pack-manager.js).
